# Shelly-s-Website
A dandy's world related browser (plain) on made Sketchwere.

**What is an Browser**: A Browser Simply contains Buttons, EditText, and an WebView, An Browser mainly focuses on Webview.

``` On sketchwere, drag this: [webview:WebView1] loadURL [text:edittext1 getText] ```

What It Does and Doesnt:
- It Does Make A First Core for a web
- It doesnt have Forward, Back, and Enter Buttons as Subcore
- It does not have a JavaScript.
