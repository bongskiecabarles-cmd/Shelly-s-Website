# Guide for the APP
## Starter Navigation.
 Press `《 and/or 》` to Navigate on the website.
 Press ` ▲ ` to Enter the Link
 Press/Edit the `github.com or shelly://` to Change the link
 Intereact the `webview1` (website) to start.
 
## shelly:// (case sensitive)

To Start: Type `shelly://help`
 - It will copy the commands that can be used.

**"all help commands copied to your clipboard"** - All Commands that is specifically Used for  The App has been Copied.
  - On Android (with connected keyboard): ` CTRL + V `
  - On Android (no keyboard) : ` Hold "github.com" 》 select "Paste" `

**"Update Logs %id.decimal%"** - The updatelogs when typed "shelly://updatelogs" (case sensitive)
  - On Android : Press Close to close.
  - On Android (Emulator on PC) : Hover over `Close` and click.
  - On Android (Emulator on VM ) : Press Close, if your VM (On Android) only supports mouse connections,then;
  - On Android (With Mouse/Emulator on VM that requires mouse connection) : Hover over `Close` and click. 

## Error
(102) - Network errors.
  - Connect to a proper network, check your router or modern connection, and reload the app*
  - Make sure your Airplane mode is off.
  - Make sure the website has proper connection, or available**

*Based on Update 5.0
**Make sure the domain is correct, it is  Letter and Numerical Sensitive
